# What is this repository

This repository is the mark that I've partially moved to another Git hosting service. Empty commits with no code information are sent from outside of Microsoft's platform via an automatized hook so that my Github profile doesn't lay dormant when I'm still coding, just not in Microsoft.

The hook is available in [this Forgejo-hosted repository](https://git.disroot.org/blyxyas/greensq), and anyone can install it with a little bit of configuration.


# Why don't you just move completely to a self-hostable forge?

My main projects (those being Rust and Clippy) are hosted on Github, with no option to interact with them from other forges. So I must remain on Github at least partially. I'm moving some of my repositories to this other instance so that a minimum of my code stays on Microsoft servers. I don't consent to AI algorithms feeding off of my code.
